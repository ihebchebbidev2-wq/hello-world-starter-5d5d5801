# Capacitor resources

Drop your raw icon and splash assets here, then run:

```bash
npx @capacitor/assets generate --android
```

Expected files:
- `icon-only.png`     — 1024×1024 transparent PNG
- `icon-foreground.png` — 1024×1024 transparent PNG (adaptive icon)
- `icon-background.png` — 1024×1024 solid color PNG
- `splash.png`        — 2732×2732 PNG
- `splash-dark.png`   — 2732×2732 PNG
